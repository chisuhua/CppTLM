## 建议工作流程

### 每次互动完成 1 个 ADR

```
互动 1: 创建 ADR-001 (TransactionContext 存储)
互动 2: 创建 ADR-002 (传播规则)
互动 3: 创建 ADR-003 (双并行实现)
...
互动 14: 创建 ADR-014 (文档结构)
互动 15-18: 整合架构规范
```

### 每个 ADR 的创建流程

1. **我提取** 讨论记录中的关键内容
2. **我草拟** ADR 文档（使用标准模板）
3. **你审查** 并提出修改意见
4. **我修订** 直到你确认
5. **标记为** Approved，继续下一个
